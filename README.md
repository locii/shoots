# Shoots
__A wordpress starter theme built on the Bones and Redux Options framework__

Bamboo is a lightweight vanilla theme modelled on the Joomlabamboo Base3 theme http://www.joomlabamboo.com/joomla-templates/base3

It supports:
* Google font integration (via the google font loader)
* Compile to Less option for all relevant theme settings
* Minimal css framework based on mobile first responsive css / HTML Boilerplate.
* Back to top button
* Lazy load images
* Sticky nav
* Mobile menu options including off canvas, toggle menu
* Advanced appearance options
* Breadcrumbs
* Custom layouts per page type
* Advanced logo options
* Author Infotmation Block
* Post format support
* Page Templates
* Google Analytics code snippets
* Import / Export settings
* Flexible widget positioning easily controlled by theme options
